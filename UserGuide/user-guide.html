How to Use
Engaging Reader

Upload a File

  1. Go to engagingreader.com
  2. Click "Upload File"
  3. Select File*
  4. Wait for AI to Convert File

Listen to a Text

  1. Press Play to start reading the text aloud.
  2. Press Pause to stop the reading at any time.
  3. Click to view reading speed options.
  4. This shows the current reading speed.
  5. Select a reading speed.
  6. The word being read is highlighted as the audio plays.
  7. The highlighted button shows the current playback state.
  8. Click the title bar to go back to Upload.

Define Words

  1. Click a word to define it.
  2. Word definitions are clear, simple, and explain the meaning in the context of the sentence.
  3. Definitions are narrated, and words are highlighted just like the main text.
  4. Click the X to return to the main text. The narration will start automatically from the word you defined.

Using your Phone

  1. Press Upload File.
  2. Select "Photo Library" to use a photo from your camera roll, or select "Take Photo" to capture a new photo.
  3. Tap a word to define it.
  4. Playback and narration work just like on the computer.
